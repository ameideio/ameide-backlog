# Enterprise Architecture Portal - High-Fidelity Wireframe

## Core Principles

1. **Single Backend Â· Unified Metamodel**  
   One API exposing processes, transformations, diagrams, documents. Every UI component consumes the same objects.

2. **Canvas-Only Editing**  
   All authoring in Canvas. Chat/dashboard widgets are read-only with "Open in Canvas" CTAs.

3. **Multi-Workspace**  
   Users switch between workspaces (e.g., "Global EA", "Retail Programme"). Each workspace = role template + widget config.
